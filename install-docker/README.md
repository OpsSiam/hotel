# Docker Installation

Install Docker CE and Docker Compose v2 on `dnf`-based systems.

## Files
- `install-docker.sh` installs Docker CE and the Compose v2 plugin

## Requirements
- Run as root
- `dnf`-based systems (e.g., CentOS/Rocky/Alma)

## Quick start
```bash
chmod +x /path/to/install-docker.sh
sudo /path/to/install-docker.sh
```
